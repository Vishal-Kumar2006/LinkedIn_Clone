<h1>💼 ProConnect – Professional Networking Platform</h1>

<p><strong>A Full-Stack LinkedIn Clone Built with MERN Stack & Next.js</strong></p>

<p>
🚀 <strong>ProConnect</strong> is a professional networking platform inspired by LinkedIn, 
built to help users connect, share professional updates, and showcase their career journey.
It is a full-stack application powered by the <strong>MERN Stack</strong> with a 
<strong>Next.js frontend</strong>, designed to simulate real-world professional networking experiences.
</p>

<h2>🚀 Getting Started</h2>

<h3>1. Clone Repository</h3>

<pre>
git clone https://github.com/Vishal-Kumar2006/LinkedIn_Clone
cd ProConnect
</pre>

<h3>2. Install Dependencies</h3>

<p><strong>Backend:</strong></p>

<pre>
cd backend
npm install
</pre>

<p><strong>Frontend:</strong></p>

<pre>
cd frontend
npm install
</pre>

<h3>3. Setup Environment Variables</h3>

<p>Create a <code>.env</code> file:</p>

<pre>
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
NEXT_PUBLIC_API_URL=your_backend_url
</pre>

<h3>4. Run Development Server</h3>

<p><strong>Backend:</strong></p>

<pre>
npm run dev
</pre>

<p><strong>Frontend:</strong></p>

<pre>
npm run dev
</pre>

<p>
Visit: <code>http://localhost:3000</code>
</p>
